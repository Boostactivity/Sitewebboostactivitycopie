# BoostActivity - Site Web Premium

Site web complet pour une agence de marketing digital, inspiré de l'excellence d'Apple.

## 🎯 Structure du Site

### Pages Principales
- **Accueil** (`/`) - Page immersive avec 11 sections riches
- **Services** (`/services`) - Vue d'ensemble des 8 services
- **Solutions** (`/solutions`) - Vue d'ensemble des 6 solutions par secteur
- **Études de cas** (`/case-studies`) - 6 cas clients détaillés
- **Tarifs** (`/pricing`) - Forfaits et services à l'unité
- **Ressources** (`/resources`) - Guides, templates et blog
- **À propos** (`/about`) - Mission, valeurs, équipe
- **Contact** (`/contact`) - Formulaire et coordonnées
- **Mentions légales** (`/legal`)

### Services (8 pages détaillées)
1. Stratégie digitale (`/services/strategie-digitale`)
2. Marketing & Acquisition (`/services/marketing-acquisition`)
3. Branding (`/services/branding`)
4. Création web (`/services/creation-web`)
5. SEO & SEA (`/services/seo-sea`)
6. Social Media (`/services/social-media`)
7. Emailing & CRM (`/services/emailing-crm`)
8. Automatisation & IA (`/services/automatisation-ia`)

### Solutions par Secteur (6 pages détaillées)
1. Restaurants & Hôtellerie (`/solutions/restaurants-hotellerie`)
2. Commerces & E-commerce (`/solutions/commerces-ecommerce`)
3. Fitness & Bien-être (`/solutions/fitness-bien-etre`)
4. Immobilier (`/solutions/immobilier`)
5. B2B & Services professionnels (`/solutions/b2b-services`)
6. Santé & Professionnels libéraux (`/solutions/sante-liberaux`)

### Études de Cas (6 projets)
1. La Belle Table - Restaurant (`/case-studies/restaurant-la-belle-table`)
2. Élégance Fashion - E-commerce (`/case-studies/boutique-mode-elegance`)
3. FitPro Gym - Fitness (`/case-studies/salle-sport-fitpro`)
4. Prestige Immobilier (`/case-studies/agence-immo-prestige`)
5. Stratégie Plus - Conseil B2B (`/case-studies/cabinet-conseil-strategie`)
6. Clinique Sourire - Santé (`/case-studies/clinique-dentaire-sourire`)

## 🎨 Design

- Design minimaliste inspiré d'Apple
- Light mode premium
- Animations fluides avec Motion/React
- Typography élégante (56-96px pour les titres)
- Espacement généreux et respirant
- Navigation avec dropdowns pour Services et Solutions

## 🛠 Technologies

- React + TypeScript
- React Router (HashRouter pour compatibilité preview)
- Motion/React pour animations
- Tailwind CSS v4
- Shadcn/ui components
- Lucide React icons
- Unsplash pour images

## 📱 Navigation

Le header inclut des **dropdowns au survol** pour :
- **Services** : Accès direct aux 8 services + lien "Voir tous"
- **Solutions** : Accès direct aux 6 solutions + lien "Voir tous"

Menu mobile responsive avec liste déroulante complète.
