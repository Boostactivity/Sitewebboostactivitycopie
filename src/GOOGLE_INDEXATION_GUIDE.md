# 🚀 Guide Complet : Faire Indexer ton Site par Google

## ✅ Étape 1 : Google Search Console (URGENT - À FAIRE MAINTENANT)

### 1.1 Ajouter ton site
1. Va sur : https://search.google.com/search-console
2. Clique sur **"Ajouter une propriété"**
3. Choisis **"Préfixe de l'URL"** et entre : `https://boostactivity.fr`
4. Clique sur **Continuer**

### 1.2 Vérifier la propriété du site
Google va te proposer plusieurs méthodes. La **plus simple** :

**Méthode DNS (via Hostinger) :**
1. Google te donne un code TXT à ajouter
2. Va dans ton Hostinger → DNS/Nameservers
3. Ajoute un enregistrement TXT avec le code fourni
4. Retourne sur Google Search Console et clique sur **Vérifier**

### 1.3 Soumettre le sitemap (IMMÉDIAT)
1. Dans Google Search Console → **Sitemaps** (menu gauche)
2. Entre : `sitemap.xml`
3. Clique sur **Envoyer**
4. ✅ Ton site sera crawlé sous 24-48h
